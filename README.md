# jcamp-data-test

provides JCAMP-DX NMR data files for testing purposes.

## usage

```js
import { Rutin } from 'jcamp-data-test';
console.log(Rutin.experiment); // to get a list
console.log(Rutin.experiment.proton); // to get a buffer with proton data
```
## acknowledgements
